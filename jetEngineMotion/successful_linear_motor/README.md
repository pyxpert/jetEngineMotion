# 双节机器人直线电机运动规划系统

## 项目概述
本项目实现了双节机器人在航空发动机/燃气轮机内腔中的直线运动规划系统，重点优化了直线电机的行程利用率。

## 核心特性
- 实现了双节机器人的蠕虫式爬行运动
- 优化了直线电机行程利用，达到7.5mm/步（利用率93.75%）
- 完整的状态码控制系统（六位数字状态码：ω1ω2ω3vm1m2）
- 包含时间参数的精确运动控制

## 状态码说明
六位数字状态码：ω1ω2ω3vm1m2
- ω1, ω2: 前、后节旋转电机状态 (0=停止, 1=正转, 2=反转)
- ω3: 中间旋转电机状态 (0=停止, 1=正转, 2=反转) 
- v: 直线电机状态 (0=停止, 1=正转, 2=反转)
- m1, m2: 电磁铁状态 (0=脱附, 1=吸附)

## 电机速度参数
- 中间旋转电机角速度：90°/s
- 直线电机动速度：15mm/s

## 运动逻辑
- 抬起前/后足：中间电机内折，正转（状态码第3位为1）
- 放下前/后足：中间电机外翻，反转（状态码第3位为2）
- 保持特定角度：中间电机停止（状态码第3位为0）
